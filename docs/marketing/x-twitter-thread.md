### Tweet 2 (How it works)
3 commands. That's it:

```
npx drift-guard init     # Lock your design
npx drift-guard rules    # Tell AI the rules
npx drift-guard check    # Catch drift
```

It snapshots your CSS tokens + DOM structure.
If AI changes your design → exit code 1 → commit blocked. 🚫
